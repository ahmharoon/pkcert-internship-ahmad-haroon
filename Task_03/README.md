# Task 03 – Python Fundamentals & Introduction to AI/ML

## Contents
| File | Covers |
|---|---|
| `part_a_python_fundamentals.py` | Part A (30): data types, functions, comprehensions, OOP/inheritance |
| `part_b_data_handling.py` | Part B (25): text file I/O, exception handling, CSV/JSON |
| `part_c_dev_environment_report.md` | Part C (10): venv vs Conda, dependency isolation |
| `part_d_ai_ml_concepts.md` | Part D (25): AI/ML/DL definitions, learning types, ML pipeline |
| `part_e_libraries_comparison.md` | Part E (10): NumPy/Pandas/Scikit-learn/TensorFlow/PyTorch comparison |
| `sample_data/` | Generated by `part_b_data_handling.py` (text/CSV/JSON output) |

## How to Run
```bash
cd Task_03
python3 part_a_python_fundamentals.py
python3 part_b_data_handling.py
```
Both scripts print their output to the console and are self-contained
(`part_b_data_handling.py` creates its own sample files under `sample_data/`
on each run). No extra installs are required — only the Python standard
library is used.
